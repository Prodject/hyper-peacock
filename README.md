# Hyper Peacock Theme

A Peacock theme for [Hyper](https://hyper.is/).

## Installation

1. Open Hyper.app preferences with your editor - Tip: `CMD+,` or at `~/.hyper.js`.
2. Add `hyper-peacock` to your plugins list.

```js
plugins: ['hyper-peacock']
```
